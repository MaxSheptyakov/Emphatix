This is a simple bot which can hepls you to start manage yourself emotions

## Technologies Used
- Python
- Redis
- Postgres
- SQL Alchemy
- docker

## Getting Started
1. Clone the repository to your PC
2. Run in workdir following command
```
sudo docker compose up --build -d
```
